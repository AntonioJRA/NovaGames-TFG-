"# NovaGames" 

## ENGLISH 🇬🇧
NovaGames is a web platform for video game management and discovery developed as a Final Degree Project (TFG).  
It allows users to explore video games, view detailed information, and manage game-related content through a modern and intuitive interface.

## Technologies

- Frontend: Angular  
- Backend: Node.js  
- Database: MySQL  
- HTML5, CSS3, JavaScript, TypeScript  
- REST API  
- Git / GitHub  

## Features

- Game listing with filtering and search functionality  
- Detailed page for each game  
- CRUD data management system  
- REST API consumption  
- Responsive design adapted to mobile devices  

## Installation

1. Clone the repository  
2. Install dependencies:
   npm install  
3. Run the project:
   npm start  

## Objective

This project was developed as part of the Higher Vocational Training program in Web Application Development (DAW), with the aim of applying full-stack development knowledge and web application architecture concepts.
